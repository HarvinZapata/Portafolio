# Autobiografica V1.0
Proyecto de diseño de página Web, con diferentes versiones. 
En este primer proyecto se construye, promera versión V1.0 con solo lenguaje HTML5.
Contenido de la página, autobiografía y portafolio. 

Proyecto personal, pensado en fortalecer los conocimientos de lo aprnedido en la plataforma Platzi, en la escuela de Desarrollo Web, se pretende construir otras dos versiones de esta misma página, añaniendo CSS V2.0 para agregar estilos y atractivo visual a la página, y JavaScript para agregar funcionalidad a la página. 

Se utilizó HTLM5 puro, se usó el control de versiones git, y obviamente GitHub para cargar el proyecto en la nube. 
En cuanto a HTML se utilizó la mayor cantidad de etiquetas para tener una página completa, sin embargo alugas de tomó la decisión de no usarlas, ya que aun no se les puede agragar interactividad con JavaScript, es algo que se va a poder ver en la versión V3.0. 
En este orden de ideas, inicialmente se buscaba en la V1.0 construir toda la estructura con HTML, pero a medida que avance el proyecto veo que se pueden profundizar cada vez mas en cada una de las tecnologías a usar (HTML, CSS, JavaScript).

Comparto mi perfil de Platzi, en donde podran evidenciar los cursos realizados, base para la construcción de este proyecto. 

Nota aclaratoría, el contenido de la página web es 100 porciento organico, no fué construido con IA, sin embargo esta herramienta si fué utilizada como material de estuido, es decir para resolver dudas, errores, y aprendizaje como ya mencioné y confirmo que es muy efectivo usarlas de forma correcta, es como tener un profesor para resolver preguntas personalizadas. 
Ejemplo: Tenía dudas en como contruir tablas, y con la IA pude comprender perfectamente como se usa.
